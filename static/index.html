RemoteDiag 사용 가이드

1. 에이전트 설치 및 실행

  1. 상단 헤더의 ⬇ woorinet_remote_diag_agent.exe 버튼으로 에이전트를 다운로드합니다.
  2. 단말기가 연결된 Windows PC에서 woorinet_remote_diag_agent.exe를 실행합니다.
  3. 에이전트는 설정된 서버로 자동 접속을 시도합니다. 접속 실패 시 3초마다 재시도합니다.
  4. 상단 에이전트 상태가 녹색으로 바뀌면 연결 성공입니다.

※ 사용 기한이 만료된 경우 에이전트 실행 시 안내 메시지가 표시되고 종료됩니다.

2. 시리얼 포트 연결

  1. 왼쪽 사이드바 시리얼 포트 (AT) 섹션에서 포트와 Baudrate를 선택합니다.
  2. 열기를 클릭합니다. 자동으로 AT 초기화 시퀀스가 실행됩니다.
  3. 초기화 완료 후 ADB 디바이스 목록이 자동으로 표시됩니다.
  4. 포트를 닫으려면 닫기를 클릭합니다. ADB 설정이 원래 값으로 복원됩니다.

※ 시리얼 포트를 먼저 연결해야 ADB 디바이스 목록이 활성화됩니다.

3. ADB 디바이스 연결

  1. 시리얼 포트 연결 후 ADB 디바이스 목록이 자동으로 표시됩니다.
  2. 장치가 하나만 연결된 경우 자동으로 선택됩니다. 여러 장치인 경우 클릭하여 선택합니다.
  3. 각 장치 항목의 닫기 버튼을 클릭하면 해당 장치의 ADB를 종료하고 목록을 갱신합니다.

4. 자동 점검

USB 모드: 시리얼 포트와 ADB 디바이스를 모두 연결한 후 ▶ 점검 시작을 클릭합니다.

네트워크 모드: 좌측 사이드바 네트워크 연결 섹션에서 단말기 IP를 연결하면 USB 없이 점검이 가능합니다.

  • IMEI 확인: 단말기 IMEI 정보 유효성 검사
  • PHONE 번호 확인: 010/012로 시작하는 번호 등록 여부
  • USIM 인식: READY(정상) / OPEN(미개통) 판별
  • RMNET_DATA IP 확인: 무선망 V4/V6 IP 할당 여부
  • BRIDGE0 인터페이스 확인: 네트워크 인터페이스 존재 여부
  • IPv4 Ping (8.8.8.8): 외부 IPv4 망 연결 확인 (평균 RTT 표시)
  • IPv6 Ping (2001:4860:4860::8888): 외부 IPv6 망 연결 확인 (평균 RTT 표시)

※ 1~5번 항목은 순서대로 실행되며, 이상 발견 시 이후 항목은 점검하지 않습니다.

※ Ping 항목(6~7번)은 1~5번 모두 정상일 때 실행되며, 두 항목 모두 결과를 확인한 후 최종 판정합니다.

※ 네트워크 연결 중일 경우 USB/ADB 연결 없이 네트워크로 모든 점검이 수행됩니다.

5. 디바이스 정보

섹션 헤더를 클릭하거나 새로고침 버튼으로 각 항목을 조회합니다.

  • 모뎀 정보: AT 명령 결과 (시리얼 포트가 열려 있어야 함)
  • 네트워크: 네트워크 인터페이스 정보
  • DNS/DHCP 정보: IP 범위, 서브넷 마스크, 임대 시간, DNS 서버
  • 메모리 정보: 메모리 사용 현황
  • 프로세스 정보: 실행 중인 프로세스 목록

6. 네트워크 연결

USB/ADB 없이 단말기에 네트워크로 직접 접속하여 모든 기능을 사용할 수 있습니다.

  1. 단말기가 Windows PC와 같은 네트워크(또는 USB RNDIS 인터페이스)에 연결되어 있어야 합니다.
  2. 좌측 사이드바 네트워크 연결 섹션에서 포트를 확인합니다. (기본값: 5002)
  3. 🔍 버튼을 클릭하면 PC 네트워크를 탐색하여 응답하는 단말기 IP를 자동으로 찾습니다.
  4. IP를 직접 입력하거나 자동 탐색 결과를 확인 후 연결을 클릭합니다.
  5. 연결 성공 시 녹색으로 상태가 표시됩니다.
  6. 이후 USB/ADB 없이 네트워크만으로 모든 기능을 사용할 수 있습니다.

※ 해제를 클릭하면 연결이 해제되고 USB 모드로 복귀합니다.

🔌 드라이버 설치 확인

단말기를 처음 연결하는 경우 아래 절차를 확인하세요.

  1. PC 장치 관리자에서 모뎀 항목을 확인합니다.
    예: Qualcomm HS-USB Modem 90DB
  2. 모뎀이 보이지 않으면, 장치를 PC에서 분리한 후 드라이버를 설치합니다.
    ⬇ Modem Driver ⬇ ADB Driver
  3. 드라이버 설치 완료 후 장치를 다시 연결합니다.
  4. 장치 관리자에서 모뎀 항목이 정상 인식되는지 확인합니다.
